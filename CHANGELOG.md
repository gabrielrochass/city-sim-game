# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
e este projeto segue [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-08

### Added
- Sistema de economia completo com orçamento, taxas e impostos
- 8 tipos de edifícios com comportamentos distintos
- Sistema de grid para construção
- Métricas de cidade (população, felicidade, crime, poluição)
- Interface de usuário completa:
  - Menu principal
  - Tela de instruções
  - HUD do jogo
  - Menu de pausa
  - Tela de game over
- Sistema de eventos global para comunicação entre sistemas
- Arquitetura limpa seguindo princípios SOLID
- Padrões de design implementados (Singleton, Observer, MVC, Factory, Strategy)
- Documentação detalhada (README, ARCHITECTURE)
- Controles básicos (SPACE para turno, ESC para pausar)
- Condições de vitória e derrota

### Features Jogabilidade
- Construir edifícios residenciais, comerciais e industriais
- Gerenciar economia da cidade
- Aumentar população através de residências
- Gerar receita com edifícios comerciais
- Aumentar felicidade com parques e escolas
- Reduzir crime com delegacias
- Monitorar poluição e impacto da indústria

### Code Quality
- Validação de dados em todas as operações
- Memory management adequado
- Null safety checks
- Encapsulamento de dados
- Documentação com XML comments
